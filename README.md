# NexPhora
Presenting NexPhora: Carrying the future forward
NexPhora is an AI-powered web-app that focuses on real-time response to critical crisis involving accidents. The reporters can book an AI-powered ambulance on the platform and send in the present condition of the patient through inputs and camera access will help the attendant get more visual inputs to the condition. The attendant will be supervising the process of updation of these inputs in the AI system that prepares a pre-diagnostic report of the patient and look up for nearby hospitals which are equipped to address the emergency. While the ambulance is on its way, the system generates basic first aid instructions based on the inputs provided that will help to utilise the first critical moments after the accident to provide basic first aid by the reporter.

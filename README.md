# NexPhora

🔗 Live Demo: [https://nexphora-1.onrender.com/](https://nexphora-1.onrender.com/)

🚀 Overview

**NexPhora** is an AI-powered web application designed to respond to critical emergency situations such as accidents in real time.

🧠 What it does

* Users (reporters) can request an AI-powered ambulance through the platform.
* They can share the patient's condition using text inputs and camera access for real-time visuals.
* An attendant monitors and updates the information in the system.
* The AI generates a **pre-diagnostic report** based on the inputs.
* It suggests **nearby hospitals** equipped to handle the emergency.
* While the ambulance is on the way, the system provides **basic first aid instructions** to help during the critical initial moments.
* 
🌟 Key Idea
NexPhora aims to reduce emergency response time and improve survival chances by combining AI-based analysis, real-time communication, and hospital coordination.
